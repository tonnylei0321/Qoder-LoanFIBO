## 列说明

| 列 | 含义 |
|---|---|
| NCC字段名 | 表中的物理字段名 |
| 中文语义 | 字段业务含义(已经 DDL 核验) |
| FIBO实体 | 该字段服务的指标所锚定的 FIBO 类(去重) |
| FIBO属性 | 对应 FIBO 对象/数据属性(去重) |
| 自定义实体 | 使用此字段的 yql 指标类 |
| 自定义属性 | 对应的 yql 属性(去重前 6 个) |
| 被引用指标 | 使用此字段的指标 ID |
